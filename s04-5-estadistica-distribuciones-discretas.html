Distribuciones de probabilidad discretas

🎯 Objetivos de esta sesión

Al final de esta sesión serás capaz de:
  • Entender el concepto de variable aleatoria y distribución de probabilidad
  • Aplicar la distribución binomial en problemas de ingeniería
  • Aplicar la distribución de Poisson para eventos raros
  • Calcular esperanza matemática y varianza de variables aleatorias

1️⃣ Variables Aleatorias y Distribuciones de Probabilidad

Una variable aleatoria discreta es una variable que puede tomar solo un número contable de valores. Su distribución de probabilidad asigna a cada valor su probabilidad de ocurrencia.

📌 Propiedades fundamentales:
  • 0 ≤ P(X = xi) ≤ 1 para todo i
  • Σ P(X = xi) = 1 (la suma de todas las probabilidades es 1)
💡 Ejemplo:

Se lanzan 3 monedas. Sea X = número de caras.

x0123
P(X=x)1/83/83/81/8

Verificación: 1/8 + 3/8 + 3/8 + 1/8 = 1 ✓

2️⃣ Esperanza Matemática y Varianza

E[X] = μ = Σ xi · P(X = xi)
Var(X) = σ² = Σ (xi − μ)² · P(X = xi)
📌 Para el ejemplo anterior (3 monedas):

μ = 0·(1/8) + 1·(3/8) + 2·(3/8) + 3·(1/8) = 0 + 0.375 + 0.75 + 0.375 = 1.5

σ² = (0−1.5)²·(1/8) + (1−1.5)²·(3/8) + (2−1.5)²·(3/8) + (3−1.5)²·(1/8)

σ² = 2.25·(1/8) + 0.25·(3/8) + 0.25·(3/8) + 2.25·(1/8) = 0.281 + 0.094 + 0.094 + 0.281 = 0.75

3️⃣ Distribución Binomial

La distribución binomial modela el número de éxitos en n ensayos independientes, cada uno con probabilidad p de éxito.

P(X = k) = C(n,k) · pk · (1−p)n−k

donde C(n,k) = n! / (k! · (n−k)!) es el coeficiente binomial.

📌 Condiciones para usar la binomial:
  • Número fijo de ensayos (n)
  • Cada ensayo tiene dos resultados: éxito o fracaso
  • Probabilidad de éxito (p) constante en cada ensayo
  • Ensayos independientes entre sí
💡 Ejemplo en ingeniería de calidad:

Una fábrica produce tornillos con un defecto del 5% (p = 0.05). Si se inspeccionan 20 tornillos al azar, ¿cuál es la probabilidad de encontrar exactamente 2 defectuosos?

n = 20, k = 2, p = 0.05

P(X=2) = C(20,2) · (0.05)² · (0.95)¹⁸ = 190 · 0.0025 · 0.3972 = 0.1887

Es decir, hay un 18.87% de probabilidad de encontrar exactamente 2 defectuosos.

🔑 Propiedades de la binomial:
  • Esperanza: E[X] = n · p
  • Varianza: Var(X) = n · p · (1−p)
  • Para el ejemplo: E[X] = 20 × 0.05 = 1 defectuoso esperado

4️⃣ Distribución de Poisson

La distribución de Poisson modela el número de eventos raros que ocurren en un intervalo de tiempo o espacio.

P(X = k) = (λk · e−λ) / k!

donde λ = número medio de eventos por intervalo (λ > 0).

📌 Cuándo usar Poisson:
  • Eventos raros (p pequeña) con muchas oportunidades (n grande)
  • Se cumple que λ = n · p (y n es grande, p es pequeña)
  • Los eventos ocurren de forma independiente
💡 Ejemplo en ingeniería:

Un servidor web recibe de media λ = 3 errores por hora. ¿Cuál es la probabilidad de que en la próxima hora no haya errores?

P(X=0) = (3⁰ · e−3) / 0! = 1 · 0.0498 / 1 = 0.0498

Solo un 4.98% de probabilidad de que no haya errores en la próxima hora.

📌 Propiedades de Poisson:
  • Esperanza: E[X] = λ
  • Varianza: Var(X) = λ
  • ¡En Poisson, la media y la varianza son iguales!

📋 Resumen de lo aprendido

  • Una variable aleatoria discreta toma valores contables; su distribución asigna probabilidad a cada valor
  • Esperanza (μ): valor esperado; Varianza (σ²): dispersión alrededor de la media
  • Binomial: B(n,p) — número de éxitos en n ensayos. E[X]=np, Var(X)=np(1−p)
  • Poisson: P(λ) — número de eventos en un intervalo. E[X]=λ, Var(X)=λ
  • La binomial se usa para defectos en n piezas; la Poisson para fallos por tiempo/espacio
  • Cuando n es grande y p es pequeña, la binomial se aproxima a la Poisson con λ = np
