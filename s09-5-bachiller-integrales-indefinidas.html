Integrales indefinidas

1. Que es una Integral

La integral es la operacion inversa de la derivada:

Si F'(x) = f(x), entonces F(x) es una primitiva de f(x).

int f(x) dx = F(x) + C

C es la constante de integracion.

3. Integracion por Sustitucion

Si u = g(x), du = g'(x)dx:
int f(g(x)) * g'(x) dx = int f(u) du
Ejemplo:

int 2x * cos(x^2) dx

u = x^2, du = 2x dx

int cos(u) du = sin(u) + C = sin(x^2) + C

4. Integracion por Partes

Formula:
int u dv = u*v - int v du

Regla LIATE para elegir u: Logaritmicas, Inversas trig, Algebraicas, Trigonomtricas, Exponenciales

Ejemplo:

int x * e^x dx

u = x, dv = e^x dx --> du = dx, v = e^x

= x*e^x - int e^x dx = e^x(x-1) + C

5. Aplicaciones

Cinematica:
  • v(t) = s'(t) --> s(t) = int v(t) dt
  • a(t) = v'(t) --> v(t) = int a(t) dt
Ejemplo:

g = 9.8 m/s^2, cae desde reposo a 100m

v(t) = 9.8t, s(t) = 100 - 4.9t^2

Toca suelo: t = sqrt(100/4.9) ~ 4.52 s

Resumen

  • La integral es la operacion inversa de la derivada
  • int x^n dx = x^(n+1)/(n+1) + C
  • Integracion por sustitucion: u = g(x)
  • Integracion por partes: int u dv = uv - int v du
  • Con primitivas se resuelven problemas cinematicos
  • Constante de integracion C siempre
