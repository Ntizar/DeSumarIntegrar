Repaso integrador de 1 Carrera

Problema 1: Gradiente y Maximo Crecimiento

Problema: f(x,y,z) = x^2*y + y*z^2. Hallar grad f(1,1,1) y direccion de maximo crecimiento.
Solucion:

f_x = 2xy, f_y = x^2+z^2, f_z = 2yz

grad f(1,1,1) = (2, 2, 2)

Direccion unitaria: (1,sqrt(3), 1/sqrt(3), 1/sqrt(3))

|grad f| = sqrt(12) = 2*sqrt(3)

Problema 2: Integral Doble en Polares

Problema: int int_R sqrt(x^2+y^2) dA, R: disco x^2+y^2<=4.
Solucion:

= int_0^{2pi} int_0^2 r * r dr dt = int_0^{2pi} dt * int_0^2 r^2 dr

= 2pi * [r^3/3]_0^2 = 2pi * 8/3 = 16pi/3

Problema 3: EDO Lineal 2 Orden

Problema: y'' - 4y' + 4y = e^(2x).
Solucion:

Ec. caract: r^2-4r+4=0 --> (r-2)^2=0 --> r=2 (doble)

y_h = (C1+C2*x)*e^(2x)

y_p: como e^(2x) y x*e^(2x) estan en y_h, probar y_p = A*x^2*e^(2x)

y_p' = A*e^(2x)*(2x^2+2x), y_p'' = A*e^(2x)*(4x^2+8x+2)

y_p''-4y_p'+4y_p = A*e^(2x)*(4x^2+8x+2-8x^2-8x+4x^2) = 2A*e^(2x) = e^(2x)

A = 1/2, y = (C1+C2*x+x^2/2)*e^(2x)

Problema 4: Autovalores y Diagonalizacion

Problema: A = [[4,1],[2,3]]. Autovalores, autovectores, A^10.
Solucion:

det(A-lI) = (4-l)(3-l)-2 = l^2-7l+10 = 0 --> l=5, l=2

l=5: v1=(1,1), l=2: v2=(1,-2)

P = [[1,1],[1,-2]], D = diag(5,2)

A^10 = P*diag(5^10, 2^10)*P^(-1)

Problema 5: Transformacion Lineal

Problema: T: R^3 -> R^2, T(x,y,z) = (x+y, y+z). Hallar nucleo, imagen, rango.
Solucion:

Matriz: [[1,1,0],[0,1,1]]

Rango = 2 (dos filas LI)

dim(ker) = 3-2 = 1

ker: x+y=0, y+z=0 --> x=-y, z=-y --> ker = span{(-1,1,-1)}
