Bachillerato - Sesion 3

Reglas de derivacion

Objetivos

Al final de esta sesion seras capaz de:
  • Aplicar las reglas basicas de derivacion (suma, producto, cociente)
  • Derivar funciones compuestas usando la regla de la cadena
  • Derivar funciones exponenciales, logaritmicas y trigonometricas
  • Calcular derivadas de orden superior

1. Reglas Basicas de Derivacion

Reglas fundamentales:
FuncionDerivada
f(x) = c (constante)f'(x) = 0
f(x) = x^nf'(x) = n * x^(n-1) (regla de la potencia)
f(x) = c * g(x)f'(x) = c * g'(x) (constante por funcion)
f(x) = g(x) + h(x)f'(x) = g'(x) + h'(x) (suma)
f(x) = g(x) - h(x)f'(x) = g'(x) - h'(x) (resta)
f(x) = g(x) * h(x)f'(x) = g'(x) * h(x) + g(x) * h'(x) (producto)
f(x) = g(x) / h(x)f'(x) = [g'(x) * h(x) - g(x) * h'(x)] / h(x)^2 (cociente)
Ejemplos:
  • f(x) = 3x^4 --> f'(x) = 12x^3
  • f(x) = 2x^3 - 5x^2 + 3x - 7 --> f'(x) = 6x^2 - 10x + 3
  • f(x) = x^2 * e^x --> f'(x) = 2x*e^x + x^2*e^x = e^x(2x + x^2)

2. Regla de la Cadena

La regla de la cadena permite derivar funciones compuestas: f(g(x)).

d/dx [f(g(x))] = f'(g(x)) * g'(x)
Pasos para aplicar la regla de la cadena:
  1. Identificar la funcion exterior f(u) y la interior u = g(x)
  2. Derivar la exterior respecto a u: f'(u)
  3. Derivar la interior respecto a x: g'(x)
  4. Multiplicar: f'(g(x)) * g'(x)
Ejemplo 1:

f(x) = (3x + 1)^5

Funcion exterior: u^5 --> derivada: 5u^4

Funcion interior: 3x + 1 --> derivada: 3

f'(x) = 5(3x + 1)^4 * 3 = 15(3x + 1)^4

Ejemplo 2:

f(x) = sin(x^2)

f'(x) = cos(x^2) * 2x = 2x*cos(x^2)

3. Derivadas de Funciones Especiales

Exponenciales y Logaritmos

FuncionDerivada
f(x) = e^xf'(x) = e^x
f(x) = a^xf'(x) = a^x * ln(a)
f(x) = ln(x)f'(x) = 1/x
f(x) = log_a(x)f'(x) = 1/(x * ln(a))
f(x) = ln(g(x))f'(x) = g'(x)/g(x)

Trigonometricas

FuncionDerivada
f(x) = sin(x)f'(x) = cos(x)
f(x) = cos(x)f'(x) = -sin(x)
f(x) = tan(x)f'(x) = sec^2(x) = 1/cos^2(x)
f(x) = arcsin(x)f'(x) = 1/sqrt(1-x^2)
Ejemplo:

f(x) = e^x * sin(x)

Usamos producto: f'(x) = e^x * sin(x) + e^x * cos(x) = e^x(sin(x) + cos(x))

4. Derivadas de Orden Superior

Podemos derivar varias veces. La segunda derivada f''(x) es la derivada de f'(x).

Interpretacion fisica:
  • s(t) --> posicion
  • s'(t) = v(t) --> velocidad (1a derivada)
  • s''(t) = a(t) --> aceleracion (2a derivada)
  • s'''(t) --> jerk (3a derivada, cambio de aceleracion)
Ejemplo:

f(x) = x^4 - 3x^3 + 2x^2 - x + 5

f'(x) = 4x^3 - 9x^2 + 4x - 1

f''(x) = 12x^2 - 18x + 4

f'''(x) = 24x - 18

f^(4)(x) = 24

Ejercicios

1. f(x) = 5x^3 - 2x^2 + 7x - 3. Cual es f'(x)?

2. f(x) = (2x + 1)^3. Cual es f'(x)?

3. f(x) = ln(x^2 + 1). Cual es f'(x)?

4. f(x) = sin(3x). Cual es f'(x)?

Resumen de lo aprendido

  • Regla de la potencia: d/dx [x^n] = n * x^(n-1)
  • Producto: (gh)' = g'h + gh'
  • Cociente: (g/h)' = (g'h - gh')/h^2
  • Regla de la cadena: d/dx [f(g(x))] = f'(g(x)) * g'(x)
  • Exponenciales: d/dx [e^x] = e^x, d/dx [a^x] = a^x * ln(a)
  • Logaritmos: d/dx [ln(x)] = 1/x
  • Trigonometricas: d/dx [sin(x)] = cos(x), d/dx [cos(x)] = -sin(x)
  • La segunda derivada f'' mide la concavidad o la aceleracion
