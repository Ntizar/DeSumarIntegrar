Algebra lineal: autovalores

1. Autovalores y Autovectores

Av: Av = l*v, donde l es autovalor, v es autovector (v != 0).
Como encontrar:
det(A - l*I) = 0 (ec. caracteristica)
Ejemplo: A = [[3,1],[0,2]]

det([[3-l,1],[0,2-l]]) = (3-l)(2-l) = 0

l1=3, l2=2

Para l=3: (A-3I)v=0 --> [[0,1],[0,-1]]v=0 --> v1=(1,0)

Para l=2: (A-2I)v=0 --> [[1,1],[0,0]]v=0 --> v2=(1,-1)

2. Diagonalizacion

A es diagonalizable si existe P invertible tal que:
P^(-1)*A*P = D (diagonal con autovalores)
Teorema: Si A tiene n autovalores distintos --> diagonalizable.
Aplicacion: A^k = P*D^k*P^(-1) (potencias faciles).

3. Espacios Propios

Espacio propio E_l = {v : Av = l*v} = Ker(A - l*I)
Multiplicidad algebraica: orden del factor (l-l_i) en el polinomio caracteristico.
Multiplicidad geometrica: dimension de E_l = n - rango(A-l_i*I).
Si mult. geo = mult. alg. para todo l --> diagonalizable.

4. Matrices Simetricas

Teorema espectral: Si A es simetrica (A=A^T):
  • Todos los autovalores son reales
  • Autovectores de autovalores distintos son ortogonales
  • A es ortogonalmente diagonalizable: A = PDP^T donde P es ortogonal

5. Aplicaciones en Ingenieria

Vibraciones: modos normales = autovectores, frecuencias = autovalores.
Google PageRank: vector propio principal de la matriz de enlaces.
PCA (ML): autovalores de la matriz de covarianza --> componentes principales.
Estructuras: analisis de estabilidad = sign de autovalores.
