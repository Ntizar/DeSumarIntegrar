🔢 1º Carrera — Sesión 8: Autovalores y Autovectores

La esencia de una matriz: sus eigenvalores y eigenvectores

🎯 Objetivos

Al final de esta sesión serás capaz de:
  • Calcular autovalores y autovectores de una matriz
  • Entender el significado geométrico
  • Aplicar a diagonalización y PageRank

1️⃣ ¿Qué son los autovalores?

1Paso 1: ¿QUÉ es un autovalor? Un autovalor λ es un número tal que existe un vector no nulo v con A·v = λ·v. Es decir, al aplicar la matriz A al vector v, solo se escala (no cambia de dirección). El vector v se llama autovector.
2Paso 2: ¿PARA QUÉ sirven los autovalores? Los autovalores revelan la "esencia" de una transformación lineal:
  • 📊 GOOGLE PageRank: el autovector dominante da la importancia de páginas web
  • 🔬 Física: frecuencias naturales de vibración de estructuras
  • 🤖 Machine Learning: PCA (Análisis de Componentes Principales) usa autovalores para reducir dimensionalidad
  • 💾 Compresión de imágenes: SVD (descomposición en valores singulares)
3Paso 3: ¿CÓMO se calculan?

Paso 1: Restar λI de A: (A − λI)

Paso 2: Calcular el determinante: det(A − λI) = 0

Paso 3: Resolver la ecuación característica para encontrar los autovalores λ

Paso 4: Para cada λ, resolver (A − λI)v = 0 para encontrar los autovectores

4Paso 4: Error común ⚠️ Confundir autovalor con elemento de la diagonal. Solo para matrices DIAGONALES los autovalores son los de la diagonal. Para matrices generales, hay que resolver det(A − λI) = 0.

💡 Truco: La suma de autovalores = traza(A) (suma de la diagonal). Es una forma de verificar tu resultado.

2️⃣ Visualización geométrica

🌍 Caso real: Google PageRank

TECNOLOGÍA Google PageRank

PageRank usa autovalores para determinar la importancia de las páginas web. Cada página web es un nodo, y cada enlace es una dirección. La matriz de enlaces tiene un autovector dominante cuyo mayor componente indica la página más importante.

¿Por qué funciona? El autovector propio dominante (el asociado al mayor autovalor) converge al estacionario del proceso aleatorio de "navegante aleatorio" que Google simula.

💡 Pista: traza = suma de autovalores, det = producto de autovalores

📋 Resumen

  • Autovalor: $A\vec{v} = \lambda\vec{v}$ — la matriz solo escala el autovector
  • Se calculan resolviendo $\det(A - \lambda I) = 0$ (ecuación característica)
  • Para matrices diagonales, los autovalores son los de la diagonal
  • La traza = suma de autovalores, el determinante = producto de autovalores
  • λ > 1 → crece, 0 < λ < 1 → se encoge, λ = 0 → se anula
  • Aplicaciones: PageRank, vibraciones, PCA, compresión de imágenes, diagonalización
  • El autovector dominante (mayor |λ|) domina el comportamiento a largo plazo
