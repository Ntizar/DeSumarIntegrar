Limites multivariable

1. Funciones de Varias Variables

Una funcion f(x,y) asigna a cada par (x,y) un valor z.
Domino: conjunto de puntos donde f esta definida.
Ejemplo: f(x,y) = x^2 + y^2 es un paraboloide. f(1,2) = 5.

2. Limites en R^2

lim_{(x,y)->(a,b)} f(x,y) = L si f(x,y) se acerca a L sin importar la trayectoria.
Para demostrar que NO existe: encontrar 2 trayectorias con limites distintos.
Ejemplo NO existe: f(x,y) = xy/(x^2+y^2) en (0,0)

Por y=0: f(x,0) = 0 --> limite 0

Por y=x: f(x,x) = x^2/(2x^2) = 1/2 --> limite 1/2

Limites distintos --> NO existe el limite.

3. Continuidad en R^2

f es continua en (a,b) si:
  • f(a,b) esta definida
  • lim_{(x,y)->(a,b)} f(x,y) existe
  • El limite coincide con f(a,b)
Teorema: Si todas las derivadas parciales existen y son continuas en un punto, la funcion es diferenciable (y por tanto continua) en ese punto.

4. Limites y Metodos

Coordenadas polares: x = r*cos(theta), y = r*sin(theta)
lim_{(x,y)->(0,0)} f(x,y) = lim_{r->0} f(r*cos(theta), r*sin(theta))

Si el limite no depende de theta, existe.

Ejemplo: f(x,y) = x^2*y/(x^2+y^2)

x = r*cos(t), y = r*sin(t)

f = r^3*cos^2(t)*sin(t) / r^2 = r*cos^2(t)*sin(t)

|f| <= r --> cuando r->0, f->0

lim = 0 (existe)

Resumen

  • Funciones de varias variables: f(x,y,z,...)
