🎯 ¿Qué vamos a aprender?

💡 Idea clave Para sumar/restar decimales, alineamos la coma y sumamos/restamos como con números enteros.

En esta sesión aprenderás a:

  • Sumar decimales
  • Restar decimales
  • Alinear la coma decimal

1️⃣ Sumar y restar decimales

📖 Teoría Alineamos la coma decimal y sumamos/restamos columna a columna. 2.5 + 1.3 = 3.8. ¡Es como sumar enteros pero con coma!
🔍 Ejemplo 1 2.5 + 1.3 = 3.8
4.75 + 2.3 = 7.05
1.25 + 0.75 = 2.0
🔍 Ejemplo 2 5.8 - 2.3 = 3.5
10.5 - 3.7 = 6.8
7.25 - 2.5 = 4.75
🔍 Ejemplo 3 3.45 + 2.67 = 6.12
8.9 - 4.35 = 4.55
15.6 + 4.4 = 20.0
