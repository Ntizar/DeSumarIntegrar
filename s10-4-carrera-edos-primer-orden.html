EDOs primer orden

1. Conceptos Basicos

EDO: ecuacion que relaciona una funcion y sus derivadas.
Orden: mayor derivada que aparece. Ejemplo: y'' + y = 0 es de 2 orden.
Solucion general: familia de funciones con constantes arbitrarias. Solucion particular: se fija la constante con condiciones iniciales.

2. Variables Separables

Forma: dy/dx = g(x)*h(y)
int dy/h(y) = int g(x) dx
Ejemplo: dy/dx = 2x*y

dy/y = 2x dx

ln|y| = x^2 + C

y = C*e^(x^2)

3. EDO Lineal de Primer Orden

Forma estandar:
dy/dx + P(x)*y = Q(x)
Factor integrante:
mu(x) = e^(int P(x) dx)
d/dx [mu*y] = mu*Q(x)
y = (1/mu) * int mu*Q dx
Ejemplo: y' + 2y = 4x

mu = e^(2x)

d/dx [e^(2x)*y] = 4x*e^(2x)

e^(2x)*y = 2x*e^(2x) - e^(2x) + C

y = 2x - 1 + C*e^(-2x)

4. EDO Exactas

Forma: M(x,y)dx + N(x,y)dy = 0
Condicion de exactitud: dM/dy = dN/dx
Solucion: existe F(x,y) tal que dF = M dx + N dy, la solucion es F(x,y) = C
Ejemplo: 2xy dx + (x^2+1) dy = 0

M_y = 2x, N_x = 2x --> exacta

F = int 2xy dx = x^2*y + h(y)

F_y = x^2 + h'(y) = x^2 + 1 --> h' = 1 --> h = y

Solucion: x^2*y + y = C --> y(x^2+1) = C

5. Aplicaciones en Ingenieria

Enfriamiento de Newton: dT/dt = -k(T - T_ambient)
Crecimiento poblacional: dP/dt = k*P --> P(t) = P_0*e^(kt)
Circuito RC: R*C*dV/dt + V = V_0
Mezclas: dA/dt = tasa_entrada - tasa_salida

Si

Resumen

  • EDO: ecuacion con derivadas de una funcion
  • Variables separables: dy/h(y) = g(x) dx
  • Lineal 1 orden: dy/dx + P*y = Q, factor integrante e^(int P)
  • Exactas: M dx + N dy = 0, condicion M_y = N_x
  • Aplicaciones: enfriamiento, poblacion, circuitos, mezclas
