# 🧮 De Sumar a Integrar

**Matemáticas para todos, desde 1º de Primaria hasta 1º de Carrera.**

Un curso digital completo, interactivo y en castellano que cubre **todo el itinerario matemático**: desde contar con los dedos hasta cálculo multivariable, álgebra lineal y ecuaciones diferenciales.

> 🚀 Generado en **un solo día** con el sistema de agentes de David Antizar en NaN.builders.

## 🎯 ¿Qué hace especial a este curso?

### 📚 Progresión real
Cada sesión conecta con la anterior y prepara la siguiente. No hay saltos bruscos. La dificultad crece de forma gradual en lenguaje, abstracción y profundidad conceptual.

### 🧠 Comprensión > Memorización
Cada capítulo responde a:
- **Qué** se hace
- **Cómo** se hace
- **Por qué** funciona
- **Cuándo** se usa
- **Qué errores** suelen cometerse

### 🎮 Interactividad con propósito
Ejercicios interactivos, visualizaciones Canvas, quizzes, problemas con solución desplegable. Nada decorativo — todo ayuda a aprender.

### 🎨 Diseño pedagógico unificado
Misma paleta (azul #2563eb + naranja #f97316), mismas cajas didácticas (teoría, ejemplo, error frecuente, idea clave), misma navegación en todas las sesiones.

### 🌐 100% en castellano
Sin anglicismos innecesarios. Lenguaje adaptado a cada etapa educativa.

## 🔍 Notas técnicas

### Para contribuir
- Las sesiones de 1º a 3º Primaria tienen CSS inline completo
- Las sesiones de 4º Primaria en adelante se generan con `generate_all.py`
- El `MEGA-PLAN.md` es el documento rector del proyecto
- Para añadir una sesión: sigue la estructura del MEGA-PLAN y el sistema de cajas didácticas

### Problema conocido
- GitHub Pages necesita `index.html` (minúsculas) para servir la raíz. El archivo principal es `INDEX.html` (mayúsculas) — ya incluimos una redirección.
