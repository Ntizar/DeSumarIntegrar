📖 CONCEPTO: Ecuaciones Diferenciales de Segundo Orden

1. ¿Qué es? Una EDO de segundo orden es una ecuación que relaciona una función y sus dos primeras derivadas: ay'' + by' + cy = f(x). Cuando f(x)=0 es homogénea.

2. ¿Para qué sirve? Modelan fenómenos con inercia y restauración: resortes, circuitos RLC, ondas, puentes, átomos. Cualquier sistema que "oscila" o "vibra" se describe así.

3. ¿Cómo se resuelve? (a) Escribe la ecuación característica: ar²+br+c=0. (b) Resuelve para r. (c) Según las raíces: reales distintas → exponentes, repetidas → ×x, complejas → sin/cos.

4. Error común: Confundir el caso de raíces repetidas (y = C₁erx + C₂xerx) con raíces distintas. ¡La x extra es la clave!

📊 Clasificación según las raíces

📽️ El discriminante Δ = b²−4ac determina el tipo de solución. ¡Memoriza los 3 casos!

🌍 Casos Reales

FÍSICA 💡 Resorte amortiguado

Un coche pasa por un bache. La suspensión oscila antes de estabilizarse. La ecuación: mx'' + cx' + kx = 0. Si c² < 4mk → oscilaciones (subamortiguado). Si c² > 4mk → vuelve sin oscilar (sobreamortiguado).

ELECTRÓNICA 💡 Circuito RLC

Un circuito con resistencia (R), inductancia (L) y capacitancia (C) oscila como un resorte eléctrico: Lq'' + Rq' + (1/C)q = 0. ¡Exactamente la misma ecuación!

INGENIERÍA 💡 Puente de Tacoma Narrows

El puente colapsó en 1940 porque el viento excitó una oscilación resonante. La ecuación del puente era una EDO de segundo orden: cuando la frecuencia del viento coincidió con la frecuencia natural, la amplitud creció hasta la destrucción.

🔗 Conexiones con otros temas

📚 Temas conectados:

• EDO de primer orden → Las de 2º orden generalizan las de 1º (una derivada → dos derivadas)

• Derivadas parciales → Las EDOs son caso particular de ecuaciones en derivadas parciales

• Series de Fourier → Las soluciones de EDOs periódicas se expanden en series de Fourier

• Espacios vectoriales → El conjunto de soluciones es un espacio vectorial de dimensión 2

• Autovalores y autovectores → Sistemas de EDOs se resuelven con autovalores de matrices

• Límites multivariable → Las condiciones iniciales en t=0 son como evaluar límites

⚠️ Errores comunes

❌ Error 1: Olvidar la x en raíces repetidas

Para y'' − 4y' + 4y = 0, muchos escriben y = C₁e²ˣ + C₂e²ˣ. ¡Incorrecto! La forma correcta es y = (C₁ + C₂x)e²ˣ. La x extra garantiza independencia lineal.

❌ Error 2: Confundir Δ con la fórmula cuadrática

Recuerda: Δ = b² − 4ac es SOLO el discriminante. No lo confundas con la fórmula r = (−b ± √Δ)/(2a). El discriminante solo te dice cuántas y qué tipo de raíces hay.

📋 Resumen

  • EDO 2º orden homogénea: ay'' + by' + cy = 0 → ecuación característica ar² + br + c = 0
  • Δ > 0 → raíces reales distintas → y = C₁eʳ¹ˣ + C₂eʳ²ˣ
  • Δ = 0 → raíz doble → y = (C₁ + C₂x)eʳˣ
  • Δ < 0 → raíces complejas α ± iβ → y = eᵅˣ(C₁cosβx + C₂sinβx)
  • Aplicaciones: resortes (mx''+cx'+kx=0), circuitos RLC, ondas, puentes
  • Las soluciones forman un espacio vectorial de dimensión 2
  • Las EDOs de 2º orden son el puente hacia ecuaciones en derivadas parciales
