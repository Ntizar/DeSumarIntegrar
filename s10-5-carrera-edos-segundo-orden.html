EDOs segundo orden

1. EDO Lineal Coeficientes Constantes

Forma: a*y'' + b*y' + c*y = 0
Ec. caracteristica: a*r^2 + b*r + c = 0
3 casos:
  • r1, r2 reales distintas: y = C1*e^(r1*x) + C2*e^(r2*x)
  • r1 = r2 = r: y = (C1 + C2*x)*e^(r*x)
  • r = a +/- bi: y = e^(ax)*(C1*cos(bx) + C2*sin(bx))

2. Caso Oscilaciones

Muelle: m*y'' + c*y' + k*y = 0
Sin amortiguamiento (c=0): y = A*cos(wt) + B*sin(wt), w=sqrt(k/m)
Amortiguado (c>0): oscilaciones que decaen exponencialmente
Sobreamortiguado: vuelve al equilibrio sin oscilar

3. EDO No Homogenea

Solucion general = y_h + y_p
y_h: solucion de la homogenea (complementaria)
y_p: solucion particular del metodo de coeficientes indeterminados
Regla para y_p:
  • Q(x) = polinomio grado n --> y_p = polinomio grado n
  • Q(x) = e^(ax) --> y_p = A*e^(ax)
  • Q(x) = sin(bx) --> y_p = A*cos(bx) + B*sin(bx)
  • Si y_p de la homogenea: multiplicar por x

4. Metodo de Variacion de Parametros

Para a*y'' + b*y' + c*y = Q(x):
Si y1, y2 son soluciones de la homogenea:
y_p = -y1 * int (y2*Q/W) dx + y2 * int (y1*Q/W) dx

W = y1*y2' - y1'*y2 (Wronskiano)

5. Aplicaciones en Ingenieria

Circuito RLC: L*q'' + R*q' + (1/C)*q = V(t)
Estructuras: m*y'' + c*y' + k*y = F(t) (vibraciones forzadas)
Transformador de corriente: ecuaciones acopladas de segundo orden
