Muestreo y estimación

🎯 Objetivos de esta sesión

Al final de esta sesión serás capaz de:
  • Entender los tipos de muestreo y cuándo usar cada uno
  • Calcular intervalos de confianza para la media
  • Calcular el tamaño de muestra necesario
  • Aplicar la distribución t de Student para muestras pequeñas

1️⃣ Tipos de Muestreo

El muestreo es el proceso de seleccionar una muestra representativa de una población. La calidad del muestreo determina la calidad de las conclusiones.

TipoCómo se haceVentajaDesventaja
Aleatorio simpleCada individuo tiene la misma probabilidad de ser seleccionadoSimple, representativoNecesita lista completa de la población
EstratificadoSe divide en estratos y se muestrea cada estratoGarantiza representación de subgruposRequiere conocer la estructura de la población
Por conglomeradosSe seleccionan grupos (conglomerados) al azarEconómico cuando la población está dispersaMenos preciso que el aleatorio simple
Consciente (por juicio)El investigador elige según criterioRápido y económicoSesgado, no representativo
💡 Ejemplo en ingeniería:

Para inspeccionar la calidad de un lote de 10,000 tornillos:

  • Aleatorio simple: numerar los 10,000 y usar generador de aleatorios
  • Estratificado: separar por turno de producción (mañana/tarde/noche) y muestrear de cada turno
  • Por conglomerados: seleccionar 10 cajas al azar de las 100 y medir todos los tornillos de esas cajas

2️⃣ Teorema Central del Límite

Este teorema es la base de toda la inferencia estadística:

📌 Teorema Central del Límite:

Si tomamos muestras aleatorias de tamaño n de cualquier población con media μ y desviación σ, la distribución de las medias muestrales se aproxima a una normal N(μ, σ/√n) cuando n es grande (n ≥ 30).

🔑 Consecuencias prácticas:
  • Las medias muestrales se distribuyen normalmente, aunque la población no lo sea
  • La desviación típica de las medias muestrales (error estándar) = σ/√n
  • A mayor n, menor dispersión de las medias → estimaciones más precisas

3️⃣ Intervalo de Confianza para la Media

Un intervalo de confianza (IC) es un rango de valores que probablemente contiene el parámetro poblacional desconocido.

IC = x̄ ± zα/2 · (σ/√n)    (si σ es conocida y n ≥ 30)
IC = x̄ ± tα/2, n-1 · (s/√n)    (si σ es desconocida o n < 30)
💡 Ejemplo en ingeniería:

Se mide la resistencia de 36 probetas de concreto. x̄ = 32.5 MPa, s = 4.2 MPa. Calcular IC al 95%.

n = 36 ≥ 30, usamos z: z0.025 = 1.96

Error estándar = s/√n = 4.2/√36 = 4.2/6 = 0.7

IC = 32.5 ± 1.96 × 0.7 = 32.5 ± 1.372

IC al 95%: [31.13, 33.87] MPa

Estamos 95% seguros de que la resistencia media real del lote está entre 31.13 y 33.87 MPa.

4️⃣ Tamaño de Muestra

¿Cuántas mediciones necesitamos para obtener un intervalo de confianza con un margen de error deseado?

n = (zα/2 · σ / E)²

donde E es el margen de error deseado.

💡 Ejemplo:

Queremos estimar la resistencia media del concreto con un margen de error de ±0.5 MPa al 95%. Sabemos que σ ≈ 4.2 MPa.

n = (1.96 × 4.2 / 0.5)² = (16.464)² = 271.1 → 272 probetas

¡Necesitamos 272 probetas! Esto muestra por qué a veces usamos muestras más pequeñas y aceptamos mayor margen de error.

📋 Resumen de lo aprendido

  • Muestreo aleatorio: cada individuo tiene igual probabilidad. Tipos: simple, estratificado, por conglomerados
  • Teorema Central del Límite: las medias muestrales se distribuyen como N(μ, σ/√n) para n grande
  • Error estándar: σ/√n (o s/√n). Disminuye con √n → más datos = más precisión
  • IC para la media: x̄ ± z · (σ/√n) o x̄ ± t · (s/√n)
  • Tamaño de muestra: n = (z · σ / E)². Mayor precisión (E pequeño) → más datos necesarios
  • IC al 95%: z = 1.96. IC al 99%: z = 2.576. Mayor confianza → intervalo más ancho
