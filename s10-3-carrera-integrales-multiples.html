Integrales multiples

1. Integral Doble

Integral doble sobre region R:
int int_R f(x,y) dA

Calcula volumen bajo z=f(x,y) sobre R.

Teorema de Fubini:
int_a^b int_c^d f(x,y) dy dx = int_c^d int_a^b f(x,y) dx dy

2. Calculo de Integrales Dobles

Region tipo I (vertical): a <= x <= b, g1(x) <= y <= g2(x)
int int_R f dA = int_a^b int_{g1(x)}^{g2(x)} f(x,y) dy dx
Region tipo II (horizontal): c <= y <= d, h1(y) <= x <= h2(y)
int int_R f dA = int_c^d int_{h1(y)}^{h2(y)} f(x,y) dx dy
Ejemplo: int_0^1 int_0^2 (x + y) dy dx

= int_0^1 [xy + y^2/2]_0^2 dx = int_0^1 (2x + 2) dx = [x^2 + 2x]_0^1 = 3

3. Cambio de Variable

Jacobiano:
J = det(d(x,y)/d(u,v)) = |x_u x_v| / |y_u y_v|
Formula:
int int_R f(x,y) dx dy = int int_S f(x(u,v), y(u,v)) * |J| du dv
Coordenadas polares: dx dy = r dr dtheta
int int_R f(x,y) dx dy = int int_S f(r cos t, r sin t) * r dr dt

4. Triple Integral

Volume en R^3:
int int int_R f(x,y,z) dV
Coordenadas cilindricas: x=r cos t, y=r sin t, z=z
dV = r dr dt dz
Coordenadas esfericas: x=r sin p cos t, y=r sin p sin t, z=r cos p
dV = r^2 sin p dr dt dp

5. Aplicaciones

Masa: M = int int_R r(x,y) dA
Centro de masas: x_bar = M_y/M, y_bar = M_x/M
Probabilidad conjunta: P((X,Y) en R) = int int_R f(x,y) dx dy
Ejemplo: Calcular volumen del cilindro x^2+y^2<=4, 0<=z<=3

V = int_0^{2pi} int_0^2 3r dr dt = 2pi * 3 * 2 = 12pi
