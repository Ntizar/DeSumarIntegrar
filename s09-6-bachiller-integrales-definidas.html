Integrales definidas y Teorema Fundamental

1. Integral Definida

La integral definida calcula el area bajo la curva:
int_a^b f(x) dx = F(b) - F(a)

donde F es una primitiva de f.

2. Teorema Fundamental del Calculo

Parte 1: Si F(x) = int_a^x f(t) dt, entonces F'(x) = f(x).
Parte 2 (regla de Barrow): int_a^b f(x) dx = F(b) - F(a) donde F' = f.

3. Calculo de Areas

Area bajo f(x) >= 0 en [a,b]:
A = int_a^b f(x) dx
Area entre dos curvas f(x) >= g(x):
A = int_a^b [f(x) - g(x)] dx
Ejemplo:

Area bajo f(x) = x^2 de 0 a 2

int_0^2 x^2 dx = [x^3/3]_0^2 = 8/3 - 0 = 8/3

4. Propiedades de la Integral Definida

PropiedadFormula
Linealidadint(a*f + b*g) = a*int f + b*int g
Aditividadint_a^b = int_a^c + int_c^b
Intervalo nuloint_a^a = 0
Cambio de ordenint_a^b = -int_b^a
Desigualdadf<=g --> int f <= int g

5. Aplicaciones en Ingenieria

Trabajo: W = int_a^b F(x) dx
Centro de masas: x_bar = (1/M) * int x*f(x) dx
Valor medio: f_med = (1/(b-a)) * int_a^b f(x) dx
Ejemplo valor medio:

Temperatura T(t) = 20 + 5sin(t/10) en [0, 60] min

T_med = (1/60) * int_0^60 (20 + 5sin(t/10)) dt

= (1/60) * [20t - 50cos(t/10)]_0^60

= (1/60) * (1200 - 50cos(6) - (-50)) = (1250 - 50cos(6))/60 ~ 20.83 grados

Resumen

  • La integral definida = area bajo la curva
  • Teorema Fundamental: int_a^b f(x) dx = F(b) - F(a)
  • Area entre curvas: int [f(x) - g(x)] dx
  • Propiedades: linealidad, aditividad, cambio orden
  • Aplicaciones: trabajo, centro de masas, valor medio
