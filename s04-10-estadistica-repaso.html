Problema 1: Estadística Descriptiva Completa

💡 Contexto:

Un ingeniero mide la resistencia a tracción (MPa) de 10 muestras de acero:

420, 435, 410, 445, 430, 425, 440, 415, 450, 432

a) Calcular media, mediana, moda, rango, varianza y desviación típica.

b) Interpretar los resultados en contexto de ingeniería.

✅ Solución:

x̄ = (420+435+410+445+430+425+440+415+450+432)/10 = 4302/10 = 430.2 MPa

Ordenados: 410, 415, 420, 425, 430, 432, 435, 440, 445, 450

Md = (432+435)/2 = 433.5 MPa

Moda: no hay (todos distintos)

Rango: 450 − 410 = 40 MPa

s² = Σ(xᵢ−430.2)²/9 = (408.04+14.44+492.84+299.24+14.44+5.76+190.44+231.04+449.44+4.84)/9 = 2160.5/9 = 240.06 MPa²

s = √240.06 = 15.49 MPa

CV = (15.49/430.2) × 100 = 3.6% → dispersión baja, proceso consistente

Problema 2: Probabilidad Binomial

💡 Contexto:

Una línea de producción tiene un defecto del 3%. Se inspeccionan 50 piezas.

a) ¿Probabilidad de que exactamente 2 sean defectuosas?

b) ¿Probabilidad de que como máximo 1 sea defectuosa?

✅ Solución:

n = 50, p = 0.03, X ~ B(50, 0.03)

a) P(X=2) = C(50,2) × (0.03)² × (0.97)⁴⁸ = 1225 × 0.0009 × 0.2303 = 0.2540

b) P(X ≤ 1) = P(X=0) + P(X=1)

P(X=0) = (0.97)⁵⁰ = 0.2181

P(X=1) = 50 × 0.03 × (0.97)⁴⁹ = 1.5 × 0.2248 = 0.3372

P(X ≤ 1) = 0.2181 + 0.3372 = 0.5553

55.53% de probabilidad de encontrar como máximo 1 defectuoso.

Problema 3: Distribución Normal

💡 Contexto:

La resistencia del concreto sigue N(35, 5) MPa. Las especificaciones exigen al menos 28 MPa.

a) ¿Qué porcentaje cumple las especificaciones?

b) ¿Qué resistencia mínima se exigiría para que solo el 5% no cumpla?

✅ Solución:

a) z = (28 − 35)/5 = −1.4

P(X ≥ 28) = P(Z ≥ −1.4) = P(Z ≤ 1.4) = 0.9192

El 91.92% del concreto cumple las especificaciones.

b) Queremos x₀ tal que P(X < x₀) = 0.05

z₀.₀₅ = −1.645

x₀ = μ + z₀ · σ = 35 + (−1.645) × 5 = 35 − 8.225 = 26.78 MPa

Problema 4: Intervalo de Confianza

💡 Contexto:

Se miden las longitudes de 25 pernos: x̄ = 10.2 mm, s = 0.3 mm.

a) Calcular IC al 95% para la media.

b) ¿Qué tamaño de muestra se necesita para que el IC tenga un margen de error de ±0.1 mm?

✅ Solución:

a) n = 25, σ desconocida → usar t con gl = 24

t₀.₀₂₅,₂₄ = 2.064

IC = 10.2 ± 2.064 × (0.3/√25) = 10.2 ± 2.064 × 0.06 = 10.2 ± 0.124

IC al 95%: [10.076, 10.324] mm

b) n = (z₀.₀₂₅ × s / E)² = (1.96 × 0.3 / 0.1)² = (5.88)² = 34.57 → 35 pernos

Problema 5: Contraste de Hipótesis

💡 Contexto:

Un fabricante afirma que sus cables tienen resistencia media de 500 MPa. Se sabe que σ = 30 MPa. Se toma una muestra de n = 36 y x̄ = 492 MPa. ¿Hay evidencia al 5% de que la resistencia es menor a 500 MPa?

✅ Solución:

H₀: μ = 500 MPa    H₁: μ < 500 MPa (unilateral izquierdo)

z = (492 − 500) / (30/√36) = −8 / 5 = −1.60

z crítico (α = 0.05, unilateral) = −1.645

−1.60 > −1.645 → No rechazamos H₀

No hay evidencia estadística al 5% de que la resistencia sea menor a 500 MPa. El resultado no es suficientemente significativo.

P valor: P(Z < −1.60) = 0.0548 → p = 0.0548 > 0.05 → no significativo

Problema 6: Correlación y Regresión

💡 Contexto:

Se estudia la relación entre temperatura de curado (°C) y resistencia del concreto (MPa):

T (°C)202530354045
R (MPa)253035384245

a) Calcular r y R²

b) Obtener la recta de regresión

c) Predecir resistencia a 32°C

✅ Solución:

x̄ = 32.5, ȳ = 35.83

b = Σ(xᵢ−x̄)(yᵢ−ȳ) / Σ(xᵢ−x̄)² = 312.5 / 175 = 1.786

a = 35.83 − 1.786 × 32.5 = −22.07

Recta: R = −22.07 + 1.786T

r = 312.5 / √(175 × 213.83) = 312.5 / 193.5 = 0.989

R² = 0.989² = 0.978 → el 97.8% de la variabilidad se explica por la temperatura

Predicción a 32°C: R = −22.07 + 1.786 × 32 = 34.87 MPa
