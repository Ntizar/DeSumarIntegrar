Bachillerato - Sesion 9

Distribuciones combinadas

1. Variables Aleatorias Conjuntas

Distribucion conjunta de X e Y:
P(X=x, Y=y) = p(x,y)

Tabla de probabilidades conjuntas.

Funcion de densidad conjunta (continua):
P((X,Y) en A) = int int_A f(x,y) dx dy

2. Independencia

X e Y son independientes si:
P(X int Y) = P(X) * P(Y)

o f(x,y) = f_X(x) * f_Y(y)

Propiedad clave:

Si X, Y independientes: E[XY] = E[X]*E[Y] y Var(X+Y) = Var(X) + Var(Y)

3. Covarianza y Correlacion

Covarianza:
Cov(X,Y) = E[XY] - E[X]*E[Y]
Coeficiente de correlacion:
rho = Cov(X,Y) / (sigma_X * sigma_Y)

rho en [-1, 1]. rho=0 = sin correlacion lineal (no necesariamente independientes).

Ejemplo:

X = rendimiento de activo A, Y = rendimiento de activo B

rho = 0.8 --> alta correlacion positiva (se mueven juntos)

rho = -0.3 --> correlacion negativa debil

rho = 0 --> sin correlacion lineal

4. Combinacion de Variables

Suma de variables:
E[X+Y] = E[X] + E[Y]
Var(X+Y) = Var(X) + Var(Y) + 2*Cov(X,Y)
Si X, Y independientes:
Var(X+Y) = Var(X) + Var(Y)
Ejemplo portafolio:

Invertir en 2 activos: X ~ N(10%, 4%), Y ~ N(8%, 9%)

Portafolio 50/50: Z = 0.5X + 0.5Y

E[Z] = 0.5*10 + 0.5*8 = 9%

Var(Z) = 0.25*4 + 0.25*9 = 3.25 (si independientes)

sigma_Z = sqrt(3.25) ~ 1.8%

5. Distribuciones Notables Combinadas

CombinacionResultado
X, Y ~ N(m,s^2) independientesX+Y ~ N(m_X+m_Y, s_X^2+s_Y^2)
X ~ B(n1,p), Y ~ B(n2,p) indep.X+Y ~ B(n1+n2, p)
X ~ Pois(l1), Y ~ Pois(l2) indep.X+Y ~ Pois(l1+l2)
X ~ Exp(l)a*X ~ Exp(l/a)

Ejercicios

1. Si X, Y indep, E[X]=3, E[Y]=5, E[XY] = ?

2. Cov(X,Y) = 4, sigma_X=2, sigma_Y=3. rho = ?

3. X ~ Pois(3), Y ~ Pois(5) independientes. X+Y ~ ?

Resumen

  • Variables conjuntas: p(x,y) y f(x,y)
  • Independencia: P(A int B) = P(A)*P(B)
  • Covarianza y correlacion: rho = Cov/(s_X*s_Y)
  • Suma de variables: Var(X+Y) = Var(X)+Var(Y)+2Cov(X,Y)
  • Combinaciones notables: Normal+Normal=Normal, Poisson+Poisson=Poisson
