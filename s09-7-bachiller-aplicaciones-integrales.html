Bachillerato - Sesion 7

Aplicaciones de integrales

1. Volmenes de Revolucion

Metodo de discos:
V = pi * int_a^b [f(x)]^2 dx
Metodo de cascarones (cilindricos):
V = 2*pi * int_a^b x*f(x) dx
Ejemplo disco:

Rotar y=x^2 de 0 a 2 alrededor del eje x

V = pi * int_0^2 (x^2)^2 dx = pi * int_0^2 x^4 dx = pi * [x^5/5]_0^2 = 32*pi/5

2. Longitud de Arco

Longitud de curva y=f(x) en [a,b]:
L = int_a^b sqrt(1 + [f'(x)]^2) dx
Ejemplo:

Longitud de y = x^(3/2) de 0 a 4

f'(x) = (3/2)x^(1/2), [f'(x)]^2 = (9/4)x

L = int_0^4 sqrt(1 + 9x/4) dx

u = 1 + 9x/4, du = 9/4 dx

= (4/9) * int_1^10 sqrt(u) du = (4/9) * [2u^(3/2)/3]_1^10 = (8/27)(10sqrt(10) - 1)

3. Centro de Masas y Momentos

Placa homogenea bajo f(x) en [a,b]:
M = int_a^b f(x) dx (masa/area)
M_x = int_a^b x*f(x) dx (momento respecto a y)
M_y = int_a^b (1/2)[f(x)]^2 dx (momento respecto a x)
x_bar = M_x/M, y_bar = M_y/M

4. Trabajo y Fuerza Hidrostatica

Trabajo (fuerza variable):
W = int_a^b F(x) dx
Fuerza hidrostatica sobre placa vertical:
F = rho * g * int_a^b h(x) * w(x) dx

h(x) = profundidad, w(x) = ancho, rho = densidad

Ejemplo trabajo:

Elevacion: F(x) = 100/(1+x) N, de x=0 a x=5m

W = int_0^5 100/(1+x) dx = 100*[ln(1+x)]_0^5 = 100*ln(6) ~ 179.2 J

5. Integrales Impropias

Intervalo infinito:
int_a^inf f(x) dx = lim_{b->inf} int_a^b f(x) dx
Discontinuidad en el intervalo:
int_a^b f(x) dx = lim_{t->b-} int_a^t f(x) dx
Ejemplo:

int_1^inf 1/x^2 dx = lim_{b->inf} [-1/x]_1^b = lim_{b->inf} (-1/b + 1) = 1 (converge)

Ejemplo:

int_1^inf 1/x dx = lim_{b->inf} ln(b) = inf (diverge)

Resumen

  • Volmenes de revolucion: V = pi * int [f(x)]^2 dx
  • Longitud de arco: L = int sqrt(1 + [f'(x)]^2) dx
  • Centro de masas: x_bar = M_x/M
  • Trabajo: W = int F(x) dx
  • Integrales impropias: limites con intervalos infinitos
